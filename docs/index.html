Lumiverse Strip

Do you want a smart light strip that integrates mutiple functions be designed to enhance users' perception, interaction, and communication, thereby improving your quality of life?

This one can help you realize your wish!

How to use it

Follow The Steps

Step 1

If you want to switch functions, you need to shout "Lumi" into this hole in the picture below.

Step 2

When you see a line like this on the box it means you've successfully awakened it.

Step 3

Then, you need to say the name of the function you want to change

Including:

  • Moving Mode
  • Music Mode
  • Environment Mode
  • Life Mode

Functions

1. MOVING MODE

In this mode, when someone passes in front of the sensor box in the picture below, the sensor detects this action and triggers the lights inside the LED box to light up column by column.

2. MUSIC MODE

In this mode, the height of each line will be modified according to the frequency and sound level of the current music (sound), acting as a musical ambient light

3. ENVIRONMENT MODE

In this mode, it will display some data about the current environment alternately, including

  • Temperature
  • Humidity
  • Pressure
  • IAQ (Indoor Air Quality)
  • Heat Index

4. LIFE MODE

There are two subfunctions in this mode

1. Weather: It will show an animation of today's weather

2. Sunrise and Sunset: When there are 30 minutes to go before sunrise and sunset, animations of the corresponding phases will be shown to simulate the real sunrise and sunset.

5. TEXT MODE

You can't wake this up by saying "Text Mode" to "Lumi". You can only send the specified message by crossing your palm over the circular hole under the sensor box. For example, when you cross your palm from left to right, it sends the message Good morning. Or download the software and send the message you want to show through the software.

PS: This is the highest priority function, and when a message is transmitted, the original function is suspended
